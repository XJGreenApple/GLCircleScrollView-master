# GLCircleScrollView
无限循环轮播图


* GLCircleScrollView 有以下主要功能：

   > 无限循环轮播
   
   > 图片点击代理
   
   > 可设置图片Url的数组（如果要赋url数组的话，最好自己改成sd加载图片的方式）


    ![](https://github.com/god-long/GLCircleScrollView/raw/master/Circle.gif) 

   
* 使用方法：

   > 下载后直接把CirCleView.swift这个文件拉进项目中即可。
   

  *这个基本上是最简单的版本了，后续会添加一些其他的功能*
